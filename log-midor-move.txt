Implemented src/services/locationService.js with the four requested exports:

getUserLocation()
reverseGeocode(lat, lng)
searchPlace(query)
getCountryCode()
Details:

getUserLocation() uses navigator.geolocation.getCurrentPosition with enableHighAccuracy: true and timeout: 10000, and resolves { lat, lng, accuracy }.
reverseGeocode() calls Nominatim reverse lookup and returns { displayName, city, country, countryCode } with Accept-Language: en.
searchPlace() calls Nominatim search and returns up to 5 mapped results as { id, displayName, lat, lng, type }.
getCountryCode() chains location + reverse geocoding and falls back to "DEFAULT" on any failure.
Error handling is graceful:

location and reverse-geocode failures return clear rejected messages where appropriate
searchPlace() returns [] on failure
getCountryCode() returns "DEFAULT" on failure
